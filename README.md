# todo-flaskapp

Setup virtual env and install flask:         
```
$ cd <app>
$ virtualenv env
$ source env/bin/activate 
$ pip3 install flask
```  

To deactivate venv:   
```
$ deactivate
```  

To run app on http://localhost:5000 
```
$ python3 app.py
```

This App is currently deployed on heroku as of 02/18/2019:
```
https://todoflaskapp.herokuapp.com/
```
